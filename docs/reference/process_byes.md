# Process bye matches automatically (round 1 only)

Process bye matches automatically (round 1 only)

## Usage

``` r
process_byes(bracket, round = 1L, bracket_type = NULL)
```

## Arguments

- bracket:

  A bracket object

- round:

  Round number to process byes for

- bracket_type:

  Optional bracket type filter

## Value

Updated bracket object with byes resolved
