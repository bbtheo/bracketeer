# Apply tiebreaker ordering

Apply tiebreaker ordering

## Usage

``` r
apply_tiebreakers(df, matches, tiebreakers, allow_ties = TRUE)
```

## Arguments

- df:

  Standings data frame with participant column

- matches:

  List of matches

- tiebreakers:

  Character vector of tiebreakers

- allow_ties:

  Logical

## Value

Ordered data frame with rank column
