# Construct a selector object for transition `take =` routing

Construct a selector object for transition `take =` routing

## Usage

``` r
new_selector(kind, params = list(), evaluator)
```

## Arguments

- kind:

  Selector kind label.

- params:

  Selector parameters list.

- evaluator:

  Function implementing selection logic.

## Value

A `bracketeer_selector` object.
